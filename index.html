Go create, install and use boilerplates with ease

Introduction

Go is a new tool for creating and using boilerplates designed to be simple to start, and fast to get it done. Go is delivered with a minimal toolset to let you immediately create boilerplates and it can be easily extended with plugins.

Quick Start

For those who like to try it immediately, install go and try our boilerplate: 

$ npm install --global go
$ go gocli/template-example new-project

You will be asked a few questions and once you answered them the folder new-project will be created with content that variats depends on your input.

What is Boilerplate?

The boilerplate is just a basic git repository. So if you already have the repository you can install it using go. Here is an example with github repository: 

$ go <username>/<repository> <destination/path>
Go supports many more options to install boilerplates! You may use different git hostings or even different boilerplate sources.

Links

Plugins

  • Cli Plugin - register and execute commands.
  • FS Plugin - simplify the work with files and folders.
  • Quiz Plugin - collect user input.
  • Handlebars Plugin - process templates using Handlebars templates.

Loaders

  • Git Loader - installs boilerplates from GitHub, Bitbucket, GitLab or any other git repository available via SSH.
  • Yeoman Loader - installs Yeoman generators.
  • VueJS Loader - installs VueJS templates.

Templates

  • gocli/template-example - an example boilerplate that shows how to build very basic boilerplate.
  • gocli/template-full-example - boilerplate that uses all features that are delivered with pure go installation.
